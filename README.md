# 📊 Student Performance Dashboard

A beginner-friendly data analysis project built with **Python**, **Pandas**, and **Matplotlib**.

This application collects student information, stores it in a CSV file, performs statistical analysis, and visualizes the data using different types of charts.

---

## 🚀 Features

- Accepts details of 5 students
- Stores data using Pandas DataFrame
- Saves records to a CSV file
- Reads data from the CSV file
- Finds:
  - Highest Marks
  - Lowest Marks
  - Average Marks
  - Students scoring above average
  - Youngest Student
- Generates multiple charts using Matplotlib:
  - 📈 Line Chart
  - 📊 Bar Chart
  - 🥧 Pie Chart
  - 📉 Histogram
  - 📍 Scatter Plot

## ▶️ How to Run

1. Clone this repository.

```
git clone <repository-link>
```

2. Install the required libraries.

```
pip install pandas matplotlib
```

3. Run the program.

```
python student_dashboard.py
```

4. Enter the details of 5 students.

5. View the generated charts and statistical analysis.
